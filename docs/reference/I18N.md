# Internationalization Notes

Factory Careers keeps Nuxt i18n plumbing and locale files from the upstream project, but the production Factory Careers experience currently defaults to English.

## Current Contract

- Default locale: `en`
- Locale source files: `i18n/locales/*.json`
- Feature flag logic: `shared/language-feature.ts`
- Nuxt config wiring: `nuxt.config.ts`
- Crowdin mapping file: `.crowdin.yml`

## Maintenance Guidance

- Keep `.crowdin.yml` only while an external translation workflow is active or expected to resume.
- Do not add a second GitHub Actions sync path unless the translation owner explicitly wants it.
- When language support changes, update `shared/language-feature.ts`, locale files, and browser coverage for the language switcher together.
