# Factory Careers Roadmap

This roadmap tracks the Factory Careers fork. Historical upstream Reqcore milestones are intentionally not repeated here.

## Current Priorities

- Harden launch operations for `careers.thefactoryhq.com`.
- Keep dashboard workflows accessible, responsive, and keyboard-complete.
- Maintain CLI parity for agent and operator automation.
- Continue tightening tenant isolation, document privacy, and production evidence.
- Keep Factory-specific configuration in environment and settings surfaces instead of hardcoding business context into shared primitives.

## Recently Completed

- Factory branding across public pages, auth flows, dashboard chrome, and email copy.
- Factory organization seed and default role setup.
- Microsoft SSO and gated dashboard access.
- Authenticated `factory-careers` CLI foundation.
- Production readiness checks, backup restore rehearsal, and object-storage rehearsal.
- Security route coverage and Playwright tenant-isolation checks.
- AI scoring context and organization-level analysis context settings.

## Backlog

- Finish production approval evidence for real candidate data.
- Keep release bundle and self-hosting docs aligned with Factory Careers image names.
- Continue dashboard polish from browser QA findings.
- Expand CLI coverage when new dashboard/API surfaces ship.
- Revisit localization only if Factory Careers re-enables multi-locale public flows.
